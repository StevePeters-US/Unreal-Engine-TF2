Quick Install & Setup

1. Download Latest Release found at https://github.com/getnamo/tensorflow-ue4-examples/releases
2. Copy to a destination of choice
3. Browse to the project folder
4. Download matching dependency found under https://github.com/getnamo/tensorflow-ue4/releases
5. Copy Plugins folder into your Project root.
6. Launch project and wait for tensorflow dependencies to download

Refer to https://github.com/getnamo/tensorflow-ue4-examples for detailed instructions
